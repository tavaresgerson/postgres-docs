## Capítulo 16. Instalação a partir de Binários

O PostgreSQL está disponível na forma de pacotes binários para a maioria dos sistemas operacionais comuns atualmente. Quando disponível, essa é a maneira recomendada de instalar o PostgreSQL para usuários do sistema. Construir a partir de fonte (consulte [Capítulo 17][(installation.md "Chapter 17. Installation from Source Code")]) é recomendado apenas para pessoas que estão desenvolvendo o PostgreSQL ou extensões.

Para uma lista atualizada de plataformas que fornecem pacotes binários, visite a seção de download no site do PostgreSQL em <https://www.postgresql.org/download/> e siga as instruções para a plataforma específica.
