## H.3. Linguagens Procedimentais [#](#EXTERNAL-PL)

O PostgreSQL inclui vários linguagens procedimentais na distribuição básica: [PL/pgSQL][(plpgsql.md "Chapter 41. PL/pgSQL — SQL Procedural Language")], [PL/Tcl][(pltcl.md "Chapter 42. PL/Tcl — Tcl Procedural Language")], [PL/Perl][(plperl.md "Chapter 43. PL/Perl — Perl Procedural Language")] e [PL/Python][(plpython.md "Chapter 44. PL/Python — Python Procedural Language")].

Além disso, há vários idiomas processuais que são desenvolvidos e mantidos fora da distribuição principal do PostgreSQL. Uma lista de [linguagens processuais][(https://wiki.postgresql.org/wiki/PL_Matrix)] é mantida no wiki do PostgreSQL. Note que alguns desses projetos não são lançados sob a mesma licença que o PostgreSQL. Para mais informações sobre cada linguagem processual, incluindo informações de licenciamento, consulte seu site e documentação.
