O PostgreSQL tem quatro formatos de documentação primários:

* Texto simples, para informações de pré-instalação
* HTML, para navegação online e referência
* PDF, para impressão
* páginas do manual, para referência rápida.

Além disso, vários arquivos `README` de texto simples podem ser encontrados em todo o conjunto de código-fonte do PostgreSQL, documentando vários problemas de implementação.

A documentação HTML e as páginas do manual fazem parte de uma distribuição padrão e são instaladas por padrão. A documentação em formato PDF está disponível separadamente para download.
