## DROP USER

DROP USER — remover um papel de banco de dados

## Sinopse

```
DROP USER [ IF EXISTS ] name [, ...]
```

## Descrição

`DROP USER` é simplesmente uma grafia alternativa de [`DROP ROLE`](sql-droprole.md "DROP ROLE").

## Compatibilidade

A declaração `DROP USER` é uma extensão do PostgreSQL. O padrão SQL deixa a definição de usuários para a implementação.
