## DROP GROUP

DROP GROUP — remover um papel de banco de dados

## Sinopse

```
DROP GROUP [ IF EXISTS ] name [, ...]
```

## Descrição

`DROP GROUP` é agora um alias para [`DROP ROLE`](sql-droprole.md "DROP ROLE").

## Compatibilidade

Não há nenhuma declaração `DROP GROUP` no padrão SQL.
