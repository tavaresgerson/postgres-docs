## G.2. Aplicações de servidor [#](#CONTRIB-PROG-SERVER)

Algumas aplicações funcionam no próprio servidor PostgreSQL. Atualmente, não há aplicações desse tipo no diretório `contrib`. Consulte também [Aplicações do servidor PostgreSQL](reference-server.md "PostgreSQL Server Applications") para obter informações sobre as aplicações do servidor que fazem parte da distribuição básica do PostgreSQL.
