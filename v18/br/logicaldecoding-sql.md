## 47.4. Decodificação lógica da interface SQL [#](#LOGICALDECODING-SQL)

Consulte a documentação detalhada sobre a API de nível SQL para interação com decodificação lógica na [Seção 9.28.6][(functions-admin.md#FUNCTIONS-REPLICATION "9.28.6. Replication Management Functions")].

A replicação síncrona (consulte [Seção 26.2.8][(warm-standby.md#SYNCHRONOUS-REPLICATION "26.2.8. Synchronous Replication")]) só é suportada em intervalos de replicação utilizados através da interface de replicação em streaming. A interface de função e as interfaces adicionais, que não fazem parte do núcleo, não suportam replicação síncrona.
