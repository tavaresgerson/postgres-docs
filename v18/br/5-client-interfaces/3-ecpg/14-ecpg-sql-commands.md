## 34.14. Comandos SQL embutidos [#](#ECPG-SQL-COMMANDS)

* [ALLOCATE DESCRIPTOR](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-allocate-descriptor.md) — alocar uma área de descritor SQL
* [CONNECT](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-connect.md) — estabelecer uma conexão com o banco de dados
* [DEALLOCATE DESCRIPTOR](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-deallocate-descriptor.md) — liberar uma área de descritor SQL
* [DECLARE](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-declare.md) — definir um cursor
* [DECLARE STATEMENT](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-declare-statement.md) — declarar o identificador de declaração SQL
* [DESCRIBE](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-describe.md) — obter informações sobre uma declaração preparada ou conjunto de resultados
* [DISCONNECT](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-disconnect.md) — encerrar uma conexão com o banco de dados
* [EXECUTE IMMEDIATE](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-execute-immediate.md) — preparar e executar dinamicamente uma declaração
* [GET DESCRIPTOR](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-get-descriptor.md) — obter informações de uma área de descritor SQL
* [OPEN](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-open.md) — abrir um cursor dinâmico
* [PREPARE](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-prepare.md) — preparar uma declaração para execução
* [SET AUTOCOMMIT](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-set-autocommit.md) — definir o comportamento de autocommit da sessão atual
* [SET CONNECTION](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-set-connection.md) — selecionar uma conexão com o banco de dados
* [SET DESCRIPTOR](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-set-descriptor.md) — definir informações em uma área de descritor SQL
* [TYPE](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-type.md) — definir um novo tipo de dados
* [VAR](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-var.md) — definir uma variável
* [WHENEVER](5-client-interfaces/3-ecpg/ecpg-sql-commands/ecpg-sql-whenever.md) — especificar a ação a ser tomada quando uma declaração SQL causa uma condição específica de classe

Esta seção descreve todos os comandos SQL específicos para SQL embutido. Consulte também os comandos SQL listados em [Comandos SQL](5-client-interfaces/3-ecpg/ecpg-sql-commands/sql-commands.md), que também podem ser usados em SQL embutido, a menos que haja indicação em contrário.
