## 17.2. Obtendo a Fonte [#](#INSTALL-GETSOURCE)

O código-fonte do PostgreSQL para as versões lançadas pode ser obtido na seção de download do nosso site: <https://www.postgresql.org/ftp/source/>. Baixe o arquivo `postgresql-version.tar.gz` ou `postgresql-version.tar.bz2` que você está interessado, e, em seguida, descompacte-o:

```
tar xf postgresql-version.tar.bz2
```

Isso criará um diretório `postgresql-version` sob o diretório atual com as fontes do PostgreSQL. Mude para esse diretório para o resto do procedimento de instalação.

Como alternativa, você pode usar o sistema de controle de versão Git; consulte [Seção I.1](git.md) para mais informações.
