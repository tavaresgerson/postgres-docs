Cada instância de um servidor PostgreSQL em execução gerencia um ou mais bancos de dados. Os bancos de dados, portanto, são o nível hierárquico mais alto para organizar objetos SQL (“objetos de banco de dados”). Este capítulo descreve as propriedades dos bancos de dados e como criá-los, gerenciá-los e destruí-los.
