# Part IV. Client Interfaces

This part describes the client programming interfaces distributed with PostgreSQL. Each of these chapters can be read independently. There are many external programming interfaces for client programs that are distributed separately. They contain their own documentation ([Appendix H](external-projects.md "Appendix H. External Projects") lists some of the more popular ones). Readers of this part should be familiar with using SQL to manipulate and query the database (see [Part II](sql.md "Part II. The SQL Language")) and of course with the programming language of their choice.
